# Apache Cordova (apache-cordova)
Apache Cordova is a mobile development framework that allows developers to build mobile applications using HTML, CSS, and JavaScript. It provides a plugin API for accessing native device capabilities like camera, GPS, contacts, and file system across iOS, Android, and other platforms.

## APIs

### Apache Cordova
Cordova provides a JavaScript API for accessing native device capabilities through plugins, a CLI for project management and building, and a plugin development API for creating custom native bridges.
